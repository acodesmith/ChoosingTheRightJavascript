What is a Single Page Application?

A single-page application (SPA), also known as single-page interface (SPI), is a web application or web site that fits on a single web page with the goal of providing a more fluid user experience akin to a desktop application.

- Wikipedia

Backbone Views

Backbone views are almost more convention than they are code — they don't determine anything about your HTML or CSS for you, and can be used with any JavaScript templating library. The general idea is to organize your interface into logical views, backed by models, each of which can be updated independently when the model changes, without having to redraw the page.

- Backbone Docs

Backbone View 


    var GarageDoor = Backbone.View.extend({

        },
        openGarageDoor: function(){
            //called on a click event from the events object
            //trigger css3 animation via dom manipulation
        },
        initialize: function() {
            this.listenTo(this.model, "change", this.render);
        },
        render: function() {
            this.$el.html( this.template( this.model.toJSON() ));
        }
    });

Imperative vs Declarative

  • Imperative programming: telling the "machine" how to do something, and as a result what you want to happen will happen.
  • Declarative programming: telling the "machine" what you would like to happen, and let the computer figure out how to do it.
- Philip Roberts
Declarative

    <p>{{ gallery }}</p>
    <img src="http://placekitten.com/g/200/300" highlight-image-gallery gallery=1>
Imperative

    app.directive("highlightImageGallery", function() {
        return {
            restrict: "A", // A = Attribute, C = CSS Class, E = HTML Element, M = HTML Comment
            link: function(scope, element, attributes) {

                element.bind("mouseenter", function() {
                    scope.gallery = "View Gallery " + attributes.gallery;
                    scope.$apply();
                });

                element.bind("mouseleave", function() {
                    scope.gallery = "Choose Gallery";
                    scope.$apply();
                });
            }
        };
    });

angular's ng-directive

ng-directives tell angular to perform certain task including but not limited to:

  • User Events - clicks, submits, mouseover

  • Parts of the DOM - img src, selected fields, styles, classes
    ngSrc, ngSelected, ngStyles, ngClass
  • Maintain Event Scope, Render Views, Include Html
    ngController, ngView, ngInclude

Javascript scope can be very confusing. When is 'this' used in an object and when to use global objects. 


    var object = function(){
        return {
            name : 'my object',
            method: function(){
                this.name = 'other name';
            },
            subObject: {
                name: 'different name',
                method: function(){
                    if(this.name=='different name'){
                        //something cool
                    }
                },
                elementFunction: function(el){

                    //el was recently refered to as this
                    el.text(this.name);
                }
            }
        }
    }

    $(function(){
        foo = new object();

        $('button').click(function(){
            var $el = $(this);

            $el.text(foo.name);

            setTimeout(function(){
                foo.subObject.elementFunction($el);
            },1000);
        });
    });

Take Aways

Single Page Applications are only growing in popularity. The massive amount of information on the internet needs to curated. Single Page Applications deliver important information to users through a seamless user interface.

  • jQuery is a great way to learn javascript, manipulation the dom and track events
  • Backbone helps to maintain large projects with a lot of DOM manipulation. It also helps with syncing data through RESTful APIs.
  • Angular provides single page application magic, but can be hard to learn. Angular handles scope, data binding and views through declarative html.
  • Javascript can be confusing?! Don't give up hope! Learn little by little and look to the open source community for help.
